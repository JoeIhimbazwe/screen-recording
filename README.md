#  Python Screen Recorder

A simple and efficient screen recording program built using **PyAutoGUI**, **OpenCV**, and **NumPy**.  
It captures your screen in real time and saves it as an `.avi` video file.  
You can easily adjust the resolution, frame rate (FPS), and save location.

---

##  Libraries Used
- `pyautogui` — for taking screenshots  
- `cv2 (OpenCV)` — for handling video writing and display  
- `numpy` — for converting images into arrays  

---

##  Features
1) Full HD screen recording  
2) Adjustable FPS (default: 30.0)  
3) Automatic save as `Recording.avi`  
4) Press **‘q’** to stop recording anytime
